- **Font Family**: System default for best compatibility

### Accessibility Features
- Voice alerts for visually impaired users

- Simple, intuitive navigation patterns
- One-tap critical actions for emergencies
- Haptic feedback for important alerts

### Localization Support
- All text available in 12 Indian languages
- Cultural adaptation of icons and colors
- Regional fraud examples and terminology
- Right-to-left language support where applicable

### User Experience Priorities
1. **Speed**: Alerts displayed within 2 seconds
2. **Clarity**: Clear, actionable information
3. **Safety**: Prominent warning indicators
4. **Simplicity**: Minimal steps to critical actions
5. **Education**: Contextual learning opportunities
